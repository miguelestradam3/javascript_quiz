# 🧠 JavaScript Quiz

A simple and interactive quiz application built with **HTML**, **CSS**, **JavaScript**, and **Bootstrap**. Players answer multiple-choice questions, receive immediate feedback, track their score, and save their high scores locally.

---

## 📸 Features

- 🎯 Multiple-choice questions
- 🔀 Randomized question order
- 📊 Progress bar showing quiz completion
- ✅ Instant feedback for correct and incorrect answers
- 🏆 High score leaderboard
- 💾 Local Storage support
- 📱 Responsive design with Bootstrap
- ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5

---

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/miguelestradam3/javascript_quiz
```

3. Open `index.html` in your browser.

No installation or dependencies are required.

---

## 🎮 How to Play

1. Start the quiz.
2. Read each question carefully.
3. Select one of the available answers.
4. Earn points for every correct answer.
5. View your final score at the end.
6. Save your name to the high score leaderboard.

---

## 📈 Scoring

- Each correct answer awards an equal number of points.
- The total possible score is **100 points**.
- Your score is automatically stored until the quiz ends.

---

## 💾 High Scores

High scores are saved using the browser's **Local Storage**, allowing your results to persist between sessions on the same device.

Example:

| Player | Score |
|--------|------:|
| Miguel | 100 |
| Sarah | 90 |
| Alex | 80 |
